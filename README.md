# Simple TODO App

Small single-page TODO application built with HTML, CSS and JavaScript.

Run locally:

1. Open `index.html` in your browser (double-click the file), or serve the folder:

```powershell
cd "c:\Users\USER\OneDrive\Documents\Programming Langauges\GitHub\ToDO"
python -m http.server 8000
# then open http://localhost:8000 in your browser
```

Features:
- Add tasks
- Toggle complete
- Delete tasks
- Clear completed
- Persistence via `localStorage`
